# Доступные команды
| Команда           | Описание                               |
| ----------------- | -------------------------------------- |
| `make client`     | Запускает клиент                       |
| `make server`     | Запускает сервер в prod режиме         |
| `make server-dev` | Запускает сервер в dev режиме          |
| `lint`            | Проверяет стилистику кода              |
| `install-deps`    | Устанавливает необходимые зависимости  |
